# WordPress Starter

## Getting Started

To initialize this project, run:
```
npm install
```

## Watch and Build
The npm scripts defined in `package.json` show the asset scripts. To watch the theme, run
```
npm run watch
```

To watch only the CSS:
```
npm run watch:css
```

To watch only the JS:
```
npm run watch:js
```


To build the assets
```
npm run build

```
To build only the CSS:
```
npm run build:css
```

To build only the JS:
```
npm run build:js
```
